# @atlasui/cli

> AtlasUI CLI — add components to your project

## Usage

```bash
# Initialize AtlasUI
npx @atlasui/cli init

# Add components
npx @atlasui/cli add button card modal

# List all components
npx @atlasui/cli list
```

## Commands

| Command | Description |
|---------|-------------|
| `init` | Set up AtlasUI in your project |
| `add <components>` | Copy components into your project |
| `list` | List all 90+ available components |
| `diff <component>` | Compare local vs latest version |

Full documentation at [AtlasUI](https://github.com/JohnDev19/AtlasUI.git).
